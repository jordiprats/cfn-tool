## cfn

AWS CloudFormation CLI tool

### Synopsis

Inspect and manage AWS CloudFormation stacks

### Options

```
  -h, --help            help for cfn
      --no-headers      Don't print headers
  -r, --region string   AWS region (uses default if not specified)
```
